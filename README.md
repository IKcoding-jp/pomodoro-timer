# ポモドーロタイマー

バニラJS（HTML / CSS / JavaScript）で作ったポモドーロタイマーです。  
JavaScript学習の一環として、自分が毎日使う道具を自作しています。

## 学習方針

**コードはすべて自分の手で書いています。**

AIツール（Claude Code）は `.claude/settings.json` の hooks 機能で `.js` `.html` `.css` ファイルへの直接書き込みをブロックする設定にしており、見本の提示と質問への回答のみに使用しています。「なぜそう書くのか」を理解してから次に進む、を徹底しています。

## 機能

- 25分カウントダウン
- スタートボタンで開始
- 分:秒 形式で表示（例：`24:09`）

## 今後追加予定

- タイマーが0になったら通知
- スタート / ストップ切り替え
- コーヒーをテーマにしたデザイン
- 残り時間を円グラフで表示するアニメーション

## 技術スタック

- HTML5
- CSS3
- JavaScript（バニラ・フレームワーク不使用）

## 動かし方

```
index.html をブラウザで開く
```
